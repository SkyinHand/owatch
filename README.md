# OverWatch CN

最新的中文守望先锋地图工坊插件, 支持语法高亮、智能提示

## 细节

> 1. 智能提示使用：打出需要自动添加的 动作或条件 的拼音，将会出现待选项，选择后可自动补全
> 2. **使用前需要将后缀名改为.watch**
> 3. 本插件的所有动态将在**[熔火工坊](https://www.owmod.net/work/view/1586)**发布，请及时查看更新内容
